🎬 Films & Clients REST API

A full REST API to manage a catalogue of films, user carts and orders. Built in Python with Docker support.

🔍 Key Features

  • CRUD operations for films and clients
  • Shopping cart logic and order processing
  • Authentication with roles: admin vs user
  • Dockerized for easier deployment

⚙️ Technologies Used

  • Python
  • Flask
  • Docker
  • REST API

📌 Learning Outcomes

Gained experience in API structuring, role-based access control, REST standards and containerization with Docker.
